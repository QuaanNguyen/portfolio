## 🧑‍💻 Run Locally

### Clone the repository

```bash
git clone https://github.com/your-username/your-portfolio.git
```
### Install dependencies

```bash
npm install
```
### Run development server locally

```bash
npm run dev
```
Open your browser and visit: http://localhost:5173
